# API Contracts (v0)

This directory contains contract-first artifacts for the Hydra backend.

## Files
- `openapi.yaml` - OpenAPI 3.1 API definition
- `schemas/register-request.schema.json` - standalone JSON Schema for register request
- `schemas/error-envelope.schema.json` - standalone JSON Schema for standard error envelope

## Notes
- Contracts follow locked decisions in `../idea.md`.
- Backend implementation should be validated against these contracts in v0 and v1.
